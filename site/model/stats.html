Documentação Estatísticas

Toda vez que o modelo já fez uma previsão para um jogo que aconteceu de verdade, dá pra conferir se ele acertou. Esta página compara as probabilidades que o modelo calculou antes de cada partida contra o resultado real dela, para todos os jogos já disputados da Série A e da Série B com um ajuste do modelo disponível.

Calibração

Se o modelo deu 60% de chance para um resultado, ele deveria acertar perto de 60% das vezes. A linha pontilhada marca esse acerto perfeito; o tamanho de cada ponto reflete quantas previsões caíram naquela faixa de probabilidade.
